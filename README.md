# AI Resume Screener

## Overview

This project is a simple AI-powered Resume Screener that compares a resume with a job description and identifies matching keywords.

## Features

* Resume keyword extraction
* Job description analysis
* Keyword matching
* Match percentage calculation

## Technologies Used

* Python
* Streamlit
* spaCy
* scikit-learn

## Files

* main.py – Main application
* keyword_matcher.py – Keyword matching logic
* resume.txt – Sample resume
* job_description.txt – Sample job description
* requirements.txt – Project dependencies
